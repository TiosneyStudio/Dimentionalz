# Dimentionalz
A tile-based 2D survival game
Yup, thats all

Controls:

W: Jump

A & D: Movement

Left Click: Destroy block

Right Click: Place block

Mouse Wheel: Use hotbar

Minimum requirements:

Computer Data Storage: 500 GB

RAM: 4GB

Processor: Intel Core i3 3.30 GHz
